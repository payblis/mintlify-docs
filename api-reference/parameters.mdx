## Text Formatting

We support most markdown formatting. Simply add `**`, `_`, or `~` around text to format it.

| **Parameter** | **Type**          | **Description**   | **Required**   |
| ------------- | ----------------- | ----------------- | ----------------- |
| Bold          | `**bold**`        | **bold**          | **bold**          |
| Italic        | `_italic_`        | _italic_          | _italic_          |
| Strikethrough | `~strikethrough~` | ~~strikethrough~~ | ~~strikethrough~~ |
